# Mounisha D | Personal Portfolio

This is a professional, single-page personal portfolio website for **Mounisha D**, a pre-final year Computer Science student specializing in full-stack web development. The portfolio showcases skills, education, projects, and contact information in a modern, responsive design with a dark premium theme.

## Features

- **Home / Introduction**
  - Hero section with name, tagline, and a full-width "View Projects" button.
  
- **About Me**
  - Extended profile summary
  - Education details
  - Certifications and achievements
  - Interests and professional goals

- **Skills**
  - Programming, web development, databases, and tools
  - Detailed descriptions of each skill category

- **Projects**
  - Detailed descriptions of four projects:
    1. Student Portfolio – Activity Points System
    2. NeuroPrompt – AI Learning Platform
    3. Eco-Friendly Shop UI
    4. Employee Salary Management System
  - Auto-flip slider with Next/Previous buttons
  - Tech stack and outcomes highlighted

- **Contact**
  - Email, phone, location
  - LinkedIn & GitHub links
  - Professional contact message

- **UI & Design**
  - Dark premium color theme (purple neon accents)
  - Scroll animations for sections
  - Responsive layout for mobile and desktop
  - Hamburger menu for mobile navigation

## Technologies Used

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js, Express.js (for projects, if applicable)  
- **Databases:** MySQL, MongoDB (for projects)  
- **Libraries/Tools:** VS Code, Postman, GitHub, NLP library
